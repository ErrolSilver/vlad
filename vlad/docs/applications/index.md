# Vlad Installed Applications

Vlad can install the following technologies:

- MySQL
- Apache2
- PHP
- Varnish
- Git
- Drush (with the 'site audit' and 'hacked!' modules)
- Munin
- Xdebug
- Ruby (via RVM)
- Sendmail
- Mailcatcher
- Memcached
- Redis
- Adminer
- XHProf
- Solr (Version 4)
- Node.js (including npm)
- ImageMagick
- PimpMyLog
